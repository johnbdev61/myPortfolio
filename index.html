Map Mash

Tools used: React, JavaScript ES6, Express, PostgreSQL and Node.js

Map Mash is an application that allows users to create their custom controller button bindings (Mashes) for their favorite video games. Users may search for other Mashes and like the ones that they find useful.

Spaced Repetition

Tools used: React, JavaScript ES6, Express, PostgreSQL and Node.js

This Spaced Repetition application helps people memorize words from a foreign language. This app will display word in one language and asks the user to recall the corresponding word in a second language. The app utilizes an algorithm that determines which words should be used to test the user.

GitHub Repo

Potty Planet

Tools used: React, JavaScript ES6, Express, PostgreSQL and Node.js

There are few things that are more daunting than potty training your first child. Potty Planet is an application that allows user to create training posts that ask users for help or advice. Other users may comment. Once the parent gets the help they need, they can click a 'Resolve' button that moves the post to a Success Stories route. Users may browse the success stories to look for helpful tips and tricks.
